1. **Clone the repository**:
```
git clone https://github.com/CyberZenDev/GhostAI.git

```

2. **Install dependencies**:
```
pnpm install
```

3. **Start the development server**:
```
pnpm run dev
```

4. Open your browser and go to `http://localhost:3000` to see **GhostAI** in action.
